# CI/CD 實戰學習計畫

> 總時長：約 3 小時
> 對象：有基本 Docker 經驗的初學者
> 前置條件：已完成 Docker 教育訓練

---

## 課程目標

- 理解 CI/CD 的概念與為什麼需要它
- 認識 GitHub Actions 的運作方式
- 能撰寫基本的 workflow 檔案
- 實作完整流程：push code → 自動 build Docker image → push 到 Docker Hub
